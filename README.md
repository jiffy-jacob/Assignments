
Module 2 Assignment

Objective
Analyse employee dataset having 458 rows and 9 columns of ABC company using Python.

Tools used

    Python
    Pandas
    Matplotlib
    Seaborn

Tasks Completed

    Data Preprocessing
    Team Analysis
    Position Analysis
    Age Group analysis
    Correlation Analysis

Insights

    - Most employees belong to New Orleans Pelicans Team.
    - Highest number of employees belong to SG position.
    - Majority employees are in age group of 22 to 26.
    - C Position receives highest salary expenditure.
    - Highest salary is for Cleveland Cavaliers team.
    - Age and Salary shows weak positive correlation.

Conclusion

 This assignment demonstrates how data analysis and visualization techniques can be used to extract meaningful insights from employee data.
